# Tips and Tricks

## Debugging

```python
import pdb 

pdb.set_trace()

n : next

l: show current code

c: continue until next set trace
```

```python
git add .
git commit -m "first commit"
git checkout *branchname*

git merge *lower_branchname*
git push origin *upper_branchname*
git pull origin *target_branchname*
```
